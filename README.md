# Feihong's PicoLisp quickstart

## Installation

### Linux

```
cd ~
wget https://software-lab.de/pil21.tgz
tar xfz pil21.tgz
rm pil21.tgz
cd pil21/src
make
echo "alias pil='cd ~/pil21 && ./pil'" >> ~/.bashrc
source ~/.bashrc
```
